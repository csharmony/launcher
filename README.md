# Harmony Launcher

This is the official repository for Harmony's launcher. 

> [!NOTE]
> Harmony is currently under development so downloading files using our API is unavailable. That makes the launcher fail when opening the game.

> [!CAUTION]
> Any version below `1.0.0` is considered **non-production**. Functionality between versions *might* change drastically!

## Forking

You are allowed to fork the launcher or use it as a base as long as you mention [heapy](https://github.com/heapyxyz) and [Harmony](https://github.com/csharmony) in credits.
